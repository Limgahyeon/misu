출처: https://themis-view.naru.pub/ + 유저 제공 PDF「신화 계보 능력 정리」(노션 내보내기, 2026-07-07)

## 세계관 개요

- **개문의 날(The Rift)**: 원인 불명의 차원 균열이 전 세계 동시 발생. 이형(하급=본능적 파괴, 상위=지능형) 출현. 같은 시점에 극소수 인간이 각성.
- **시간축**: 원년 개문 → 1~2년차 각국 각성자 쟁탈전 후 THEMIS 설립 → 3~4년차 안정화 → 현재 5~7년차 (표면적 안정, 내부 갈등 심화).
- **센티넬**: 전투형 각성자. 압도적 신체능력. 능력 사용 시 정신 마모 → 침식 → 균열 → 폭주 → 전락(비가역).
- **가이드**: 센티넬의 폭주를 막을 유일한 존재. 신체 접촉으로 에코 공명, 정신 안정화. 전투 능력 없음. 전략적 우선 표적.
- **에코**: 각성 순간 각인되는 근원적 힘의 파편.
- **계보**: 올림포스(질서·보호·빛, 가장 흔함) / 티탄(원시력·본능, 잠재력 높음) / 타르타로스(혼돈·죽음, 폭주 위험 최고) / 프로토게노이(원초 존재 4명뿐: 카오스·닉스·가이아·아이테르) / 크세노스(그리스·로마 외 신화).
- **결속(Bond)**: 센티넬-가이드 1:1 전속 매칭. 공명율 최상. 몸에 영구 흔적. 파트너 사망 시 흔적 소멸, 센티넬 폭주 위험 최고조.
- **THEMIS 조직**: 최고 수장 Council(프로토게노이 4인). 센티넬 직급 OLYMPIAN(최정예) > PHALANX(현장 주력) > DORU(일반) > NEOPHYTE(신입). 가이드는 독립 부서 MUSE(S~E등급), 최고 가이드팀 VESTAL. 특수팀 ZERO(폭주 진압·전락자 무력화), SPECTER(이탈자 추적·정보).
- **한국 본부 TKH**: 서울 용산구, 지하 7층~지상 50층 타워. 센티넬 400, MUSE 200, 행정 600.
- **THE FORSAKEN**: 이탈자(자아 온전, 자발적 이탈) / 전락자(이형에게 각인 덮어쓰임. 공식적으론 비가역·무력화 대상이나 실제로는 일부 자아 유지 가능) / 이형(차원 너머 존재, 인간이었던 적 없음. 위협 등급 하급~특급).
- **압주(APSU)**: 버려진 자들이 모이는 숨겨진 도시. THEMIS도 위치를 모름. 규칙은 등가교환. 관리자가 도시 전체 에코 시그니처를 심연의 격으로 은폐. 위로 올라갈수록 격이 높음. 이성 유지제(에코 출력 유지하며 자아만 붙들어 매는 약물) 유통.
- **핵심 긴장**: THEMIS의 통제·질서 vs 압주의 등가교환·생존. 전락자의 자아 회복 가능성. 에코의 근원에 대한 의문.

## 신화 계보 능력 정리 (PDF 전사)

형식 — 센티넬: 주능력 / 부능력 / 폭주 양상. 가이드: 주능력 / 부능력 / 폭주 대응.

### 올림포스

**아레스** — S: 신체 강화 극대화, 싸울수록 강해짐 / 본능적 전투 예지 / 폭주: 전투 외에도 파괴 충동, 피아 구분 흐려짐. G: 전투 광기 버프 주입(고통 차단) / 광역 사기 고양 / 폭주 대응: 파괴 충동을 자신에게 흡수(가이드 신체 부담).

**아폴론** — S: 광선 수렴·방사(열선/섬광) / 예지 편린(수초 앞, 제어 불가) / 폭주: 눈에서 광선, 시야 백색, 주변 소각. G: 시야 공유·실시간 전술 전달 / 단기 예지 경보 / 폭주 대응: 음악적 파장 안정화, 목소리만으로도 일부 효과.

**아르테미스** — S: 감각 극대화(수 km 표적 포착) / 잠행 특화(존재 은폐) / 폭주: 감각 역폭주, 모든 자극이 고통. G: 감각 동기화·과부하 필터링 / 아군 은폐 보조 / 폭주 대응: 접촉 시 완충재 역할.

**헤르메스** — S: 순간 가속·공간 도약 / 접촉 물체 순간이동 투척 / 폭주: 의지와 무관한 랜덤 이동, 신체 분열 환각. G: 이동·반응 속도 증폭 / 전장 통신 중계 / 폭주 대응: 좌표 고정, 접촉으로 위치 강제 안정화.

**헤파이스토스** — S: 화염·용암 생성 조작, 열 면역 / 구조물 즉석 제작 / 폭주: 체온 통제 불능, 접촉물이 녹음. G: 고열 방어막(물리·열 흡수) / 장비 임시 수복 / 폭주 대응: 과열 에코 냉각·흡수(화상 부담 전이).

**아테나** — S: 에너지 배리어(반사·흡수 후 역방사) / 전술 연산 가속 / 폭주: 배리어 전방위 확장, 아군도 차단. G: 전술 정보 직접 주입 / 팀 행동 조율 / 폭주 대응: 과부하 연산을 자신의 파장으로 정리, 냉정함 전이.

**포세이돈** — S: 수압 조작·해일 생성 / 지진 유발 / 폭주: 몸에서 물이 새어나옴, 수분 폭발적 팽창. G: 상태이상 전체 정화(독·화상·마비) / 치유 촉진 / 폭주 대응: 수압 흐름을 자신의 파장으로 흡수 유도.

**디오니소스** — S: 혼돈 유발(환각·공황 강제) / 통증 역치 극대화(광란 상태) / 폭주: 혼돈 파장이 아군에게도 번짐. G: 정신적 혼란·공포·고통 흡수(폭주 억제 최상위 효율) / 적에게 혼돈 역주입(가이드 중 유일한 공격형) / 폭주 대응: 혼돈을 받아내며 센티넬 고정.

**니케** — S: 승리 본능(열세일수록 강해짐) / 단거리 비행·날개 에코 / 폭주: 승리 집착, 아군 희생 무감각. G: 전투력 가변 증폭(위기일수록 극대화) / 팀 사기·집중력 / 폭주 대응: 전투 외 감각 주입으로 의식 되돌림.

**에로스** — S: 만물 인력 조작(중력 유사) / 에코 공명 감지 / 폭주: 인력 역전, 블랙홀처럼 끌어당김. G: 에코 공명 인위적 극대화(결속 효율 한계 초과) / 비결속 센티넬과도 임시 공명 / 폭주 대응: 에코 인력으로 폭주 파장 흡착·분산.

**헤카테** — S: 결계 생성·파괴 / 저주 부여(에코 부식) / 폭주: 결계에 갇히는 환각, 차원 경계 너머 이형 접근. G: 차원 경계 강화(이형 침투 차단) / 저주·상태이상 해제 / 폭주 대응: 타르타로스 계열 폭주에 특화, 경계 봉인해 복귀 유도.

**이리스** — S: 무지개 빛 굴절 다리(광속 이동) / 빛 은폐막 / 폭주: 빛 난반사 통제 불능, 랜덤 착지. G: 원거리 에코 연결 유지(거리 무제한 가이딩) / 팀 정보 채널 / 폭주 대응: 연결 과부하 시 이중 감각 진입.

**티케** — S: 확률 조작 / 행운 감지 / 폭주: 확률 역전, 최악의 결과로 수렴. G: 행운 파장 주입(회피·명중 향상) / '재시도' 파장(팀 임계 상황 1회 되돌림) / 폭주 대응 리스크: 행운 역전 시 파트너에게 불운 집중.

**아스클레피오스** — S: 독소 면역·역이용 / 자기 재생 / 폭주: 재생 폭주, 조직이 뒤틀리게 자람. G: 치유 전문 가이딩(회복 속도 최상위) / 독소·부식·저주 즉시 정화 / 폭주 대응 리스크: 치유 파장 과잉 분비.

**아프로디테** — S: 감각 침식(특정 감각 잠식·무력화) / 매혹 파장(판단력·전의 약화) / 폭주: 자신의 감각이 역침식, 내면 함몰. G: 감각 증폭·정밀화 / 접촉 시 강렬한 안정감(공명 효율 최상위급) / 폭주 대응: 내면 함몰형에 특화, 감각 침식 역전.

### 티탄

**프로메테우스** — S: 불꽃 생성·조작(창의적, 출력은 헤파이스토스보다 낮음) / 지식 직관(적 능력 분석) / 폭주: 불꽃이 자신을 태움, 멈출 수 없음. G: 능력의 창의적 응용 유도 / 팀 지식 공유 / 폭주 대응: 고통 루프 감각 회로 차단.

**헬리오스** — S: 태양 에너지 수렴·방사(광역·지속) / 빛으로 공간 스캔(은폐 탐지) / 폭주: 체표 발광 제어 불능, 주변 소각. G: 전장 조명(아군 시야·적 은폐 무효) / 태양 에너지 충전(체력·에코 회복) / 폭주 대응: 장기형 안정화.

**크로노스** — S: 국소 시간 조작(늦추기·되감기) / 반응 속도 시간 압축 / 폭주: 시간 감각 붕괴(과거·현재·미래 동시). G: 폭주 진행 속도 지연 / 쿨다운 회복 가속 / 폭주 대응: 현재 시점으로 의식 고정.

**에오스** — S: 새벽빛 파장(동상·경직·수면 강제 해제) / 광역 조명 / 폭주: 빛이 주변을 맹목으로 만듦(자신 포함). G: 아군 상태이상 각성 / 에코 재점화 / 폭주 대응: 빛 과부하 흡수·소거.

**히페리온** — S: 중력 조작(증폭·무효) / 천체 감각 / 폭주: 중력 요동, 공중 부양/지면 압착. G: 중력 면역 부여 / 아군 이동 보조 / 폭주 대응: 지면 고정, 물리적 위치 안정화.

**테티스** — S: 물 흐름 완전 제어(수중 특화, 물 무기 성형) / 조류 감지 / 폭주: 신체 액화 환각. G: 흐름 공명(에코 출력 최적화) / 광역 회복 파장 / 폭주 대응: 흐름 정돈, 형태 회복 유도.

**아틀라스** — S: 압도적 물리 내구(팀의 방패) / 대지 고정 / 폭주: '혼자 다 감당해야 한다'는 강박, 아군을 밀어냄. G: 물리 내구 강화·피해 대신 흡수 / 팀 경직 면역 / 폭주 대응: 접촉으로 '혼자가 아님'을 에코에 전달.

**므네모시네** — S: 기억 열람·삭제 / 과거 기술 흡수 / 폭주: 타인 기억과 뒤섞임, 자아 붕괴. G: 폭주 패턴·심리 균열 기억 열람으로 맞춤 가이딩 / '평온했던 기억' 주입 / 폭주 대응: 자신의 기억을 닻으로 파트너를 현재로 인도.
